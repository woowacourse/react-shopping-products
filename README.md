## 3. 상품 장바구니 담기

- [x] 담기 버튼을 누르면, 장바구니에 추가된다. 이 때 장바구니에 담긴 아이템 '종류' 의 개수로 숫자를 표시한다.
- [x] 장바구니 담기 요청 중 에러가 발생한 경우, 에러 메시지를 사용자에게 알려주는 UI를 표시한다.
- [x] 장바구니에서 빼기 버튼을 누르면, 장바구니에서 해당 아이템이 제거된다.
- [ ] 장바구니에 담긴 아이템 종류의 최대 개수는 50개이다.
